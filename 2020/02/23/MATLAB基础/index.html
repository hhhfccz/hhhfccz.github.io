MATLAB基础知识

系统环境

若命令行过长需要在第一行末尾添加续行符’…’来表示需要继续读取下一行的内容

遇到同名的变量名与内部函数名时，优先使用变量名

在写程序时，应该避免变量名与内部函数名重合

数值数据

整形

分无符号整数与带符号整数
分别分为8位（二进制）整数（uint8/int8）、16位整数、32位整数、64位整数

如果赋值时，所赋值超过对应数据类型范围，按最大值记

浮点型

分单精度(single)与双精度型(double) 其中数值数据默认为double型

复数型

分实部(real)与虚部(imag)，均默认为double型，虚数单位用i/j（用j是为了与其他语言如python共用）表示

format格式符

决定数据的输出格式
单条format指short格式（小数点后四位）

该函数不影响数值本身，仅影响输出

常用数学函数

  • sqrt(n) n的平方根
  • exp(n) 求e的n次方
  • sin(弧度)或sind(角度) 求sin()值
  • abs() 求实数绝对值、虚数的模、字符串的ASCII码
  • fix() 舍弃小数取整
  • floor() 向下取整
  • ceil() 向上取整
  • round() 四舍五入取整
  • isprime(n) 判断n是否为素数（是，则返回1；不是则返回0）
  • rem(n/m)或mod(n/m) 取余

寻找1到100中的所有素数

变量及其操作

  • 变量名必须以字母开头

  • 在语句末尾加;就意味着不输出最终结果

  • 预定义变量，是由MATLAB本身定义的变量 ans默认赋值变量 pi=π NaN代表非数

  • who()显示所有变量名 whos()显示所有变量名及其对应属性

  • 使用.mat文件保存MATLAB工作区的变量

矩阵

矩阵是MATLAB中最基本的对象

矩阵的建立

行向量建立

矩阵元素的引用

通过下标来引用矩阵的元素

a(2,3) 指a矩阵第2行第3列的元素 可以通过赋值语句修改或拓展矩阵的值

通过矩阵元素的序号来引用在MATLAB中矩阵按列存储，以第一行第一列为第一个元素，第二行第一列为第二个元素……（序号从1开始，而不是从0开始）

序号与下标可以通过D = sub2ind(S行数和列数组成的向量（用size(A)获取）,I转换元素的行下标,J转换元素的列下标)（下标转序号）与[I,J]=ind2sub([a,b]a行b列的矩阵,[n]第n个元素的下标)转换

冒号运算符

  • A(:) 将矩阵堆叠为一维数组时
  • A(i, :) 第i行的全部元素
  • A(:, j) 第j列的全部元素
  • A(i:i+m, k:k+m) 第i~i+m行加上第k~k+n列的元素
  • A(i:i+m, :) 第i~i+m行的全部元素

end表示对应维度的最末尾的元素下标

空矩阵

没有任何元素的矩阵

将对应的矩阵赋值为空矩阵就意味着删除对应矩阵

矩阵形状改变

  • reshape(A,m,n) 将A矩阵重新排列为m行n列的矩阵，存储顺序以MATLAB序号为准
  • A(:) 将A矩阵压缩为一个列向量，存储顺序不变

关系运算

＜ ＞ ≤ ≥ == ~=（不等于）

标量比较，返回结果为0/1
同型矩阵之间比较（或者是标量与矩阵的比较），对应元素逐个比较，逐个赋值0/1

逻辑运算

&与 |或 ~非

字符串处理

字符串就是用单引号括起来的字符序列，MATLAB中将一行字符串当成一个行向量

字符串中的单引号用两个单引号表示（不是双引号）

当然也可以形成一个字符串矩阵

字符串执行

  • eval(s) 将s代表的字符串中的内容用对应的MATLAB命令来执行

得到’0’这个字符串

字符串与数值之间的转换

  • char() 将ASCII码矩阵转换为字符串矩阵

字符串的比较

长度相同运用关系运算符进行比较，运算方法参考矩阵之间的比较

大小评判标准为字符对应的ASCII码

专用的字符串比较

  • strcmp(s1, s2) 比较两个字符串是否完全相等，是则返回1，非则返回0
  • strncmp(s1, s2, n) 比较两个字符串前n个字符是否完全相等，是则返回1，非则返回2
  • strcmpi(s1, s2) 比较两个字符串在忽略大小写的前提下是否相等，是则返回1，非则返回2
  • strncmpi(s1, s2, n) 比较两个字符串前n个字符在忽略大小写的前提下是否相等，是则返回1，非则返回2

字符串的查找与替换

  • findstr(s1,s2) 返回短字符串在长字符串中的开始位置
  • strrep(s1,s2,s3) 将字符串s1中的所有子字符串s2替换为s3

MATLAB矩阵处理

特殊矩阵

通用的特殊矩阵及调用格式

  • zeros() 产生全0矩阵
  • ones() 产生全1矩阵
  • eye() 产生主对角线为1的矩阵
  • rand() 产生(0,1)区间均匀分布的随机矩阵
  • randn() 产生平均值为0，方差为1的标准正态分布随机矩阵

调用格式（以zeros为例）

  • zeros(m) 产生m行m列的全0矩阵
  • zeros(m,n) 产生m行n列的全0矩阵
  • zeros(size(A)) 产生与矩阵A同型的全0矩阵

产生有一定范围的随机整数n阶矩阵A

产生平均值为μ，方差为σ^2的随机n阶矩阵A

用于专门学科的特殊矩阵

  • 魔方矩阵 A = magic(n) 生成一个特定的n阶方阵，每行每列每个对角线上的元素之和均相等(n + n^3) / 2

  • 范德蒙矩阵 A = vander(v) 生成一个以v向量为基础的范德蒙矩阵

  • 希尔伯特矩阵A = hilb(n) 生成一个n阶希尔伯特矩阵，其规律为矩阵的第i行第j列的元素为1/(i+j-1)

  • 伴随矩阵 A = compan(p) 生成一个以p向量为多项式系数的多项式的伴随矩阵

  • 帕斯卡矩阵 A = pascal(n) 生成一个n阶帕斯卡矩阵，其规律为第i行第j列元素的值为第i行第j-1列元素与第i-1行第j列元素的和

矩阵变换

对角阵

  • 对角矩阵：只有对角线上有非0元素的矩阵

  • 数量矩阵：对角线上元素相等的对角矩阵

  • 单位矩阵：对角线上元素都为1的对角矩阵

    提取矩阵的对角线元素 diag(A,k)提取A第k条对角线的元素，产生一个列向量，k默认为0

    构造对角矩阵 diag(V,k)以向量V为第k条对角线的元素，产生对角矩阵，k默认为0

    矩阵每行乘一个向量用左乘，矩阵的每列乘一个向量用右乘

三角阵

上三角阵：triu(A,k) 提取矩阵A第k条对角线及以上的元素，k默认为0

下三角阵：tril(A,k) 提取矩阵A第k条对角线及以下的元素，k默认为0

矩阵的转置

转置： A.’

共轭转置： A’

共轭转置仅针对复数矩阵，实数的共轭转置与转置结果相同

矩阵的旋转

逆时针旋转： rot90(A,k)将矩阵A逆时针旋转90°的k倍，k默认为1

矩阵的翻转

左右翻转： fliplr(A) 将矩阵左右翻转
上下翻转： flipud(A) 将矩阵上下翻转

对矩阵的副对角线求和等同于将原矩阵上下翻转，求翻转后矩阵的主对角线元素之和

矩阵的逆

矩阵的逆： inv(A)

用于线性方程组求解：A*x = b => x = inv(A) * b

矩阵求值

矩阵的行列式 det(A)

性质：det(inv(A)) = 1 / det(A)

矩阵的秩 rank(A)

矩阵的迹 trace(A)

性质：trace(A) = sum(diag(A))

矩阵的范数

向量的三种范数

  • 向量1范数：向量元素的绝对值之和 norm(V,1)
  • 向量2范数：向量元素绝对值的平方和的平方根 norm(V,2)或者norm(V)
  • 向量∞范数：所有向量元素绝对值中的最大值 norm(V,inf)

矩阵的三种范数

  • 矩阵1范数：矩阵列元素绝对值之和的最大值 norm(A,1)
  • 矩阵2范数：矩阵A’A的最大特征值的平方根 norm(A,2)或者norm(A)
  • 矩阵∞范数：所有矩阵行元素绝对值之和的最大值 norm(A,inf)

矩阵的条件数（对应矩阵的三种范数）

关于矩阵范数对应的条件数： cond()

其调用格式与范数相同

矩阵的特征值与特征向量

  • E = eig(A) 求矩阵A的全部特征值，组成向量E
  • [X,D] = eig(A) 求矩阵A的全部特征值，构成对角阵D，产生矩阵X，X的第i列是D中第i列元素对应的特征向量
  • eigshow() 演示单位圆上的向量x和A*x的关系

稀疏矩阵

完全存储 ：将矩阵的所有元素按列存储起来
稀疏存储 ：稀释存储方式仅存储矩阵中的非0元素的值及其行列序号，当然，仍然按列存储

稀疏存储

两种存储方式之间的转化

  • A = sparse(S) 将S转化为稀疏存储方式的矩阵A
  • S = full(A) 将A转化为完全存储方式的矩阵S

直接建立稀疏存储矩阵

  • sparse(m,n) 生成一个m行n列的全0的稀疏存储矩阵
  • sparse(u,v,S) u,v,S分别是三个等长的向量，分别代表行下标、列下标、要存储的非0元素值
  • spconvert(A) 将A表示的矩阵直接转换为稀疏存储矩阵其中A矩阵第一行表示非0元素的行下标、第二行表示非0元素的列下标、第三行表示非0元素的实部、第四行表示非0元素的虚部

带状稀疏矩阵

带状系数矩阵，即仅对角线上有非0元素的矩阵

  • [B,d] = spdiags(A) 将A中全部非0对角线元素赋给B，对角线位置（也就是指是第几条对角线存在非0元素）向量保存在d中
  • A = spdiags(B,d,m,n) m,n分别为原矩阵B的行数与列数，B的第i列即为A的第i条非0对角线，d表示位置向量

单位矩阵的稀疏存储

  • speye(m,n) 生成一个m行n列的单位稀疏存储矩阵

求最大元素与最小元素

max() / min()

求取向量或矩阵的最大/小元素位置（若包含复数元素，按模求取最大值）

y = max(X) / [y, k] = max(X)

返回向量X的最大值存入y，最大值对应的序号存入k；返回矩阵X的每一列的最大值存在y中（或增加参数dim = 1），最大值对应的行号存在k中（或增加一个参数dim = 2）

如果要求取行上的最大值，将向量转置即可
如果要求整个矩阵的最大值，连续两次的max()即可，或者用eig()或A(:)将矩阵堆叠为一维

多项式计算

多项式的表示

n次多项式用一个n+1的行向量表示
如：p(x) = [an,an-1,……,a0] 表示a的n次方、n-1次方……

多项式系数向量的顺序是从高到低
若存在某项多项式系数不存在，用0补足

多项式的四则运算

直接 + -
* ：conv(P1, P2) 两个多项式系数向量相乘
/ ：[Q, r] = deconv(P1, P2) 两个多项式系数向量相除 Q为商，r为余

conv()与deconv() 互成逆函数 即P1 = conv(Q, P2) + r

多项式的求导

polyder() 多项式的求导函数

  • p = polyder(P) 求多项式P的导函数
  • p = polyder(P, Q) 求P·Q的导函数
  • [p, q] = polyder(P, Q) 求P/Q的导函数，导函数分子存入p，分母存入q

多项式的求值

向量多项式

polyval(p, x) 代数多项式求值
p为多项式系数向量，x可以是标量、向量、矩阵

  • x为标量，则求多项式在该点的值
  • x为向量或矩阵，则对向量/矩阵中的每一个元素求多项式的值（矩阵点乘）

矩阵多项式

polyvalm(p, x) 矩阵多项式求值
x必须为方阵，以方阵为自变量求多项式的值（矩阵乘法）

多项式的求根

  • roots(p) 求解高次多项式的根，包括实根与共轭复根
  • poly(x) 根据多项式的全部根，求取多项式 x = roots(a)

一维数据插值
